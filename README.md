GreenGrocer Data Analysis:
Overview:
This project analyzes business performance for a fictional grocery store chain called GreenGrocer. It uses Python to explore sales data, customer behavior, store performance, and basic forecasting models.

What This Project Does:
Generates simulated sales, customer, and store data
Analyzes sales trends across stores, departments, and time periods
Examines customer segments and spending behavior
Identifies relationships between business metrics (correlation analysis)
Builds a simple predictive model for store sales
Produces business insights and recommendations

Files Included:
greengrocer_analysis.py – Main Python script containing data generation, analysis, and visualizations

Tools Used:
Python
Pandas
NumPy
Matplotlib

Key Insights:
Sales vary significantly across stores and departments
Miami and Tampa are top-performing locations
Produce and Prepared Foods generate higher margins
Customer segments strongly influence spending behavior
Seasonal trends impact overall sales performance

What I Learned:
How to work with large datasets using Pandas
Creating visualizations with Matplotlib
Performing descriptive, diagnostic, and predictive analysis
Understanding business metrics and performance indicators
Turning raw data into actionable insights

Summary:
This project demonstrates how data analysis can be used to understand business performance and support decision-making in a retail environment.
